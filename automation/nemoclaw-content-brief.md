### NVIDIA NemoClaw
- **Là gì**: Nền tảng mã nguồn mở cho enterprise AI agents
- **Công bố**: GTC 2026, keynote Jensen Huang ngày 16/3/2026, San Jose
- **Đặc điểm chính**:
  - Open-source, hardware-agnostic (chạy được trên GPU không phải NVIDIA)
  - Built-in security & privacy (khác OpenClaw đã bị OpenAI mua)
  - Tích hợp NeMo framework + Nemotron models + NIM (NVIDIA Inference Microservices)
  - Target: customer service, supply chain, multi-step task automation
- **Partners tiềm năng**: Salesforce, Cisco, Google, Adobe, CrowdStrike
- **Thị trường agentic AI**: dự kiến $28B vào 2027
- **Event đặc biệt**: "Build-a-Claw" — workshop customize AI assistants tại GTC
- **Chiến lược**: NVIDIA mở rộng từ hardware sang software layer, tạo standard cho enterprise AI

### NVIDIA Nemotron 3 Super (bổ sung ngữ cảnh)
- **Launch**: 11/3/2026
- **Specs**: 120B parameters, 12B active (MoE), 1M token context window
- **Kiến trúc**: Hybrid Mamba + Transformer + MoE routing
- **Hiệu năng**: 5x throughput vs Nemotron Super trước, 4x memory efficiency
- **Open weights**: license permissive, cho phép customize + deploy tự do
- **Benchmark**: PinchBench 85.6%, top DeepResearch Bench
- **Users**: Perplexity, Amdocs, Palantir, Cadence, Dassault, Siemens
- **Ứng dụng**: software dev, cybersecurity triaging, deep research
