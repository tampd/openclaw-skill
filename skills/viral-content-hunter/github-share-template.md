```
[Mô tả ngắn gọn 2-3 dòng: repo này làm gì, tại sao đáng chú ý]

📌 Điểm nổi bật:
🚀 [Feature/điểm hay 1]
⚡ [Feature/điểm hay 2]
🧠 [Feature/điểm hay 3]

⭐ [Số stars] stars trên GitHub
👉 [URL GitHub repo]

Bạn đã thử [tên tool/repo] chưa? Dùng thế nào rồi kể nghe nào 👇

#TýTech #GitHub #OpenSource #[hashtag chủ đề]
```

## Quy tắc viết

1. **LUÔN** bắt đầu bằng "Ai đó đã chia sẻ..." hoặc biến thể tương tự cho repos/resources
2. **LUÔN đính kèm ≥1 hình ảnh** — screenshot, banner, infographic, hoặc meme
3. **KHÔNG** copy mô tả gốc từ GitHub — viết lại bằng giọng Tý Tech
4. **LUÔN** kèm link GitHub gốc
5. **Emoji**: 4-6 emoji đa dạng, mỗi bullet dùng icon khác nhau (🚀⚡🧠🎯🔧💎)
6. **Kaomoji**: Thỉnh thoảng xen kaomoji cho bài vui: (☞ﾟヮﾟ)☞  ¯\_(ツ)_/¯  (╯°□°)╯
7. **CTA**: Kết bằng câu hỏi mở + 👇 hoặc 🫡
8. **Hashtags**: 3-5 tags, luôn có `#TýTech`
9. **Credit**: Mention tên tác giả/org nếu biết
10. **Ngắn gọn**: Toàn bộ bài ≤ 500 ký tự (trừ hashtags)
